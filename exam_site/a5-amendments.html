A5-5 Question 5 of 45

Which of the following cryptography options increases overhead and cost?

A Using symmetric encryption instead of asymmetric encryption.
B Using a long asymmetric encryption key.
C Encrypting the hash instead of the message.
D Using a secret key.
Correct Answer: Option B

Explanations

A Asymmetric algorithms require more processing time than symmetric algorithms.
B Using longer asymmetric encryption keys increases computer processing time significantly. For instance, doubling an RSA key length from 512 bits to 1,024 bits can cause the decryption time to increase nearly six-fold.
C Encrypting a hash requires less overhead compared to encrypting the entire message, as the hash is usually shorter than the original message.
D A secret key used in symmetric encryption is generally small and is specifically used for encrypting user data, resulting in lower overhead.
A5-6 Question 6 of 45

Which of the following measures is an effective method for handling a data loss event experienced by an enterprise?

A Sharing sensitive data via unencrypted email
B Allowing unrestricted access to sensitive data
C Implementing regular data backups
D Using weak passwords for user accounts
Correct Answer: Option C

Explanations

A Sharing sensitive data via unencrypted email is not an effective method for preventing data loss. Email communication is susceptible to interception and unauthorized access. Without encryption, data can be easily compromised, leading to data loss or unauthorized disclosure.
B Allowing unrestricted access to sensitive data is not an effective method for preventing data loss. It increases the risk of data breaches and unauthorized access. Implementing access controls and limiting access to authorized individuals can reduce the potential for data loss or misuse.
C Regular data backups are an effective method for preventing data loss. By creating frequent backups, enterprises can ensure they can restore data to its previous state in the event of data loss or corruption. This practice helps protect against accidental data deletion, hardware failure, and data breaches.
D Using weak passwords for user accounts is not an effective method for preventing data loss. Weak passwords are easily guessable or susceptible to brute-force attacks, making user accounts vulnerable to unauthorized access. Strong passwords, along with additional security measures such as multifactor authentication, are crucial for protecting sensitive data.

A Control risk
B Inherent risk
C Detection risk
D Sampling risk
Correct Answer: Option A

Explanations

A Automated controls are generally more effective than manual controls for data loss prevention. Therefore, using manual controls results in a higher control risk compared to automated controls.
B Inherent risk refers to the risk present before any controls are implemented. The use or absence of controls does not impact inherent risk.
C Detection risk is the risk that an auditor will not detect weaknesses in the control system. This is not applicable to the scenario of choosing between manual and automated controls.
D Sampling risk is related to the risk of making incorrect conclusions about a population based on a sample. It does not apply to the context of manual versus automated controls for data loss prevention.
